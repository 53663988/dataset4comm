# dataset4comm
AI Dataset for communication/telecommunications/network or else

专注电信、通信、网络智能化相关的数据集
从电信的建设运维阶段分为：规划、建设、维护、优化、运营、资管等系统和阶段流程。

从专业域方面分为：无线、核心、承载、固网、IDC、MEC、跨域等环节。

从AI任务领域分为：检测、识别、分类、预测、决策、优化等可能交叠的方法。
